# Quick Start Guide

## Installation & Setup

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Development Server
```bash
npm run dev
```

The game will automatically open at `http://localhost:5173`

### 3. Build for Production
```bash
npm run build
```

Output will be in the `dist/` folder

## Game Features

### Money Page
- Click "Earn Money" button to generate $1 per click
- View your current balance and total earnings
- Track your click count

### Investments Page
- Browse available investment opportunities ($100-$600)
- Each investment generates daily returns
- Risk levels affect potential returns (⭐⭐⭐)
- Click "Refresh" to see new opportunities
- View your active investments

### Businesses Page
- Browse business opportunities ($200-$1,200)
- Each business generates hourly income
- View total hourly income from all businesses
- Click "Refresh" to see new opportunities
- View your active businesses

### Owner Profile
- View complete portfolio overview
- Track net worth (money + investments + businesses)
- See passive income statistics
- Monitor total clicks and investments

## Game Mechanics

### Earning Money
- **Clicking**: $1 per click
- **Hourly Bonus**: Random $100-$1,100 every UTC hour
- **Investments**: Daily returns (5-15% of investment cost)
- **Businesses**: Hourly income (2-8% of startup cost)

### Progression
1. Start by clicking to earn money
2. Once you have $100+, invest in opportunities
3. Once you have $200+, start businesses
4. Build passive income streams
5. Grow your net worth

### Saving
- Game automatically saves to browser local storage
- Your progress persists between sessions
- No account required

## Tips for Playing

1. **Early Game**: Focus on clicking to accumulate initial capital
2. **Mid Game**: Start with small investments and businesses
3. **Late Game**: Build multiple income streams
4. **Strategy**: Balance between saving and investing
5. **Optimization**: Check the Owner page to track your net worth

## Troubleshooting

### Game Not Loading
- Clear browser cache
- Try a different browser
- Check browser console for errors

### Save Not Working
- Check if local storage is enabled
- Try clearing browser data
- Restart the browser

### Performance Issues
- Close other browser tabs
- Disable browser extensions
- Try a different browser
